# Денис — Portfolio

Личный сайт-портфолио. Деплоится на GitHub Pages.

## Структура
- `index.html` — сама страница
- `assets/avatar.jpg` — аватарка
- `assets/projects/elovpark.jpg` — превью проекта
- `assets/projects/sibstalker.jpg` — превью проекта

## Как задеплоить

1. Создать новый публичный репозиторий на GitHub: `username.github.io`
2. Скопировать туда всё содержимое этой папки (index.html + assets/)
3. Через 1-2 минуты сайт будет доступен на `https://username.github.io`

## Локальная проверка
Открыть `index.html` в браузере.

## Редактирование
- Имя, контакты, описание — в `<header class="hero">`
- Стек, услуги — в соответствующих `<section>`
- Проекты — добавить новый `<div class="project">`, положить превью в `assets/projects/`
